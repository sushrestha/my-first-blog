Web Security Puzzles

These are web security puzzles to help reinforce what is needed for secure web applications. These puzzles go from very simple to more complex and modern examples.

Level 0 Information

Level 0 is a simple login page that has a simple solution to break. Solve Level 0 to get access to Level 1.

Level 1 Information

Level 1 is another login page that has a simple solution to break, with something a little new thrown into it. Solve Level 1 to get access to Level 2.

Level 2 Information

Level 2 is built on top of Level 1 but again has a few new features that are going to require little more thought. Solve Level 2 to get access to Level 3.
